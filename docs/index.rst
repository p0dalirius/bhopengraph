A Python library to create BloodHound OpenGraphs easily.

This module provides Python classes for creating and managing graph structures that are compatible with BloodHound OpenGraph. The classes follow the `BloodHound OpenGraph schema <https://bloodhound.specterops.io/opengraph/schema>`_ and `best practices <https://bloodhound.specterops.io/opengraph/best-practices>`_.

If you don't know about BloodHound OpenGraph yet, a great introduction can be found here: `https://bloodhound.specterops.io/opengraph/best-practices <https://bloodhound.specterops.io/opengraph/best-practices>`_
